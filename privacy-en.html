Privacy Policy

RefiTrak - Refrigerant Inventory Management App

1. General Information

RefiTrak was created with user privacy in mind. All data entered into the app is stored exclusively locally on the user's device.

2. What data do we collect?

The app may collect the following information:

  • PIN code - stored locally, used to secure access to the app
  • Recovery email - optional, stored locally
  • Refrigerant data - stock levels, purchases, orders - stored locally
  • Photos - added to orders, stored locally on the device
  • GPS location - optional, only when user grants permission, stored locally
  • Device country/region - automatically retrieved from device settings for statistical purposes

3. How do we use the data?

All data is used exclusively for app functionality:

  • Managing refrigerant inventory
  • Recording orders and purchases
  • Generating annual reports
  • Exporting documentation

4. Data Sharing

We do not share any data with third parties. All data remains on the user's device.

Users can export data (e.g., to CSV or PDF) and share it at their own discretion.

5. Security

Access to the app is protected by a PIN code set by the user. Data is stored in a local database on the device.

6. App Permissions

The app may request the following permissions:

  • Camera - for taking documentation photos
  • Gallery - for selecting existing photos
  • Location - optional, for adding GPS to orders
  • Storage - for saving exported files

7. Data Deletion

Users can delete all data at any time by uninstalling the app or clearing the app data in device settings.

8. Contact

For questions regarding this privacy policy, please contact:

Email: refiltrackstudio@gmail.com

9. Changes to Privacy Policy

We reserve the right to make changes to this privacy policy. Any changes will be communicated through app updates.
